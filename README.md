#  Stock market - app for watching stocks rate

## General info
The main idea of the application is to show the stock price chart and brief information about the application

## Technologies
Project is created with:
* Kotlin
* Compose
* Compose destinations
* Opencsv
* Retrofit
* Room
* Coroutines
* Coil
* Hilt
* MVI
